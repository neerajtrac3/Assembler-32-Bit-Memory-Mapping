📘 Assembler 32‑Bit Memory Mapping
Technical Publication Authored by Neeraj Aggarwal
This repository contains the original technical publication authored by Neeraj Aggarwal on Assembler 32‑Bit Memory Mapping, originally published in Infosys KShop (Document ID: 515640).

The document provides a comprehensive, end‑to‑end explanation of OS/390 Assembler memory architecture, addressing a critical knowledge gap for engineers working on legacy systems, debugging, and modernization programs.

🔹 Purpose of the Publication
This work was created to serve as a single, authoritative reference for:

24‑bit, 31‑bit, and 32‑bit addressing

OS/390 memory layout and addressing modes

PSA, TCB, ASCB, and control block structures

Storage protection and key management

Register conventions and save/restore logic

Debugging memory‑related defects

Understanding program flow in Assembler

At the time of authorship, no consolidated reference existed for these topics, making this document a foundational learning asset.

## 🧠 Key Topics Covered

- OS/390 memory architecture  
- 24‑bit, 31‑bit, and 32‑bit addressing  
- PSA, TCB, ASCB, and control blocks  
- Storage protection & key management  
- Register conventions and save/restore logic  
- Memory debugging and traceability  
- Program flow and addressing modes
